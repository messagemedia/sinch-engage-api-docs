# MessageMedia API

Welcome to the MessageMedia API documentation. This API provides powerful messaging capabilities for sending and receiving SMS, MMS, and rich messages.

## Authentication

The API supports two authentication methods:
- **Basic Authentication**
- **HMAC Signature Authentication**

## Base URLs

- **EU instance**: `https://eu.app.api.sinch.com`
- **US instance**: `https://us.app.api.sinch.com`
- **Singapore instance**: `https://sg.app.api.sinch.com`
- **Australia instance**: `https://au.app.api.sinch.com`

## Rate Limiting

API requests are subject to rate limiting. Please refer to the specific endpoint documentation for rate limit details.

## Error Handling

All error responses follow a consistent format with appropriate HTTP status codes and error details in the response body.
